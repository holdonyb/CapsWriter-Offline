# Changelog

## v2.5 (2026-03-17)

### Bug 修复

- **音频缓冲区积压**：修复说完话后结果不出来、再按一次才触发的问题（录音开始时自动清空残留队列）
- **异步死锁**：修复 keyboard 操作在 async 上下文中偶发卡死（改用 `to_thread()` 包装）
- **润色前文无响应**：修复选中文字后说"润色前文"无任何反馈的问题（增加 clipboard 延迟 + "润色中…"提示）
- **语音替换不稳定**：修复"把X换成Y"只搜索最近一次输出、说了其他话就失效的问题（改为搜索最近 5 条输出）
- **PyInstaller 工作目录**：修复 exe 打包后工作目录指向 `internal/` 导致找不到模型文件（使用 `sys.executable` 定位 exe 目录）
- **子进程重复启动**：修复 `multiprocessing` 在 frozen 环境下导致服务端 banner 反复打印（加入 `freeze_support()` + `__main__` 守卫）

### 新功能

- **语音清理（LLM 驱动）**：默认角色自动去除填充词（嗯/啊/呃）、口吃重复、识别自我纠正，全部由 LLM 智能判断
- **录音状态指示器**：录音时右下角显示脉冲动画，识别完成短暂显示结果后淡出
- **应用感知语调**：LLM 自动获取当前活跃窗口的应用名称，据此调整输出风格
- **格式化角色**：说"格式化"将口述文字整理成 Markdown 列表/步骤格式
- **只读查询命令**：`问一下`/`总结一下`/`解释一下`/`翻译一下` 对选中文字操作，结果在 Toast 弹窗显示
- **语音点击 UI 元素**：说"点击 XXX"模糊匹配当前窗口按钮并点击（`uiautomation` 已内置打包）
- **语音快捷操作**：`搜索 X`、`打开 X`、`截图`、`复制这段`
- **方向键与光标控制**：`上`/`下`/`左`/`右`（支持N次）、`Ctrl左`/`Shift下` 等组合键、`行首`/`行尾`/`文档开头`/`文档结尾`
- **删除指定词**：`删除XXXX` 从最近 5 次输出中找到并删除指定词
- **撤销/重做**：`撤销`（Ctrl+Z）、`重做`（Ctrl+Y）
- **切换窗口**：说"切换窗口"进入 Alt+Tab 模式，上下左右选择，说"确定"切换、"取消"放弃

### 改进

- 完善 PyInstaller 打包配置（`build.spec` / `build-client.spec`），支持 Python 3.11
- 新增 `runtime_hook.py` 确保 exe 旁的用户代码优先加载
- 默认角色 system prompt 全面重写，涵盖口语清理示例
